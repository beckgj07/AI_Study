# AI_Study 页面设计文档

## 设计系统

| 属性 | 值 |
|------|-----|
| 风格 | Claymorphism（软陶风格） |
| 主色 | #2563EB（学习蓝） |
| 辅色 | #3B82F6 |
| 强调色 | #F97316（活力橙） |
| 背景色 | #F8FAFC |
| 文字色 | #1E293B |
| 字体 | Baloo 2 / Comic Neue |

---

## 页面列表

| # | 页面 | 路由 | 说明 |
|---|------|------|------|
| 1 | 登录/切换账号 | `/login` | 选择孩子账号 |
| 2 | 首页仪表盘 | `/dashboard` | 概览今日学习 |
| 3 | 知识地图 | `/map` | 可视化闯关进度 |
| 4 | 闯关选择 | `/challenge` | 选择章节闯关 |
| 5 | 答题测验 | `/quiz/[id]` | 答题+即时阅卷 |
| 6 | AI 讲解 | `/explain/[id]` | 错题讲解+讨论 |
| 7 | 错题本 | `/wrong` | 错题记录+巩固 |
| 8 | 学习报告 | `/report` | 详细数据报告 |
| 9 | 成就墙 | `/achievements` | 徽章成就展示 |
| 10 | 家长端 | `/parent` | 家长管理功能 |
| 11 | AI 配置 | `/admin/ai-config` | 模型配置管理 |
